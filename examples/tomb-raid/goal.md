# 盗墓 · 任务目标（Mission Goal）

本场任务：一支 **4 人** 摸金小队完成一次 **倒斗**（盗墓）行动。

- **DM（主持人 / 上帝）**：按 `dm.md` 主持，私下分配身份、描绘场景、操纵机关与粽子、裁定玩家行动结果，直到队伍取得宝物撤离，或团灭于墓中。
- **玩家**：按 `player.md` 扮演 DM 私下分配的身份（摸金校尉 / 卸岭力士 / 搬山道人 / 发丘中郎将），与队友协作完成任务。

胜利条件（**满足任一条**）：
- **大胜**：取得目标宝物，全员活着出墓。
- **惨胜**：取得目标宝物，但有队员折在墓中。

失败条件（**满足任一条**）：
- 全员阵亡。
- 目标宝物被毁、被夺、或被永久封禁于墓中。
- 触怒墓主未能脱身（如惊动镇墓兽且无人活着抵达出口）。

由 DM 宣布"风起灯灭"后正式入墓。任务结束时，由 DM 公布所有暗线（墓主身份、隐藏机关、未触发的禁忌），并宣告战果。
